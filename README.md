# WeCare-Inventory-Sales-Management-System

## How to Run the Project

### Prerequisites
Make sure you have Python 3.x installed on your system. You can download it from [python.org](https://www.python.org/).

### Running the Application
1. **Clone or Download the repository:**
   - Click the green **Code** button above and select **Download ZIP**, then extract it.
   - Alternatively, clone it via Git:
     ```bash
     git clone https://github.com/3-sujal/WeCare-Wholesale.git
     ```
2. **Navigate to the project directory:**
   ```bash
   cd WeCare-Inventory-Sales-Management-System
   ```
3. **Execute the program::**
    ```bash
   python main.py
   ```
